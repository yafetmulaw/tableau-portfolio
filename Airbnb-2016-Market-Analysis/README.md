# Airbnb 2016 Market Analysis

Interactive Tableau dashboard analyzing Airbnb listings data from 2016.

## Objective
Understand how pricing and revenue vary by property size and location.

## Dashboard Highlights
- Average price by number of bedrooms
- Geographic price distribution by zipcode
- Weekly revenue trends and seasonality

## Key Insights
- Larger properties command significantly higher prices
- Certain zipcodes consistently outperform others
- Revenue shows steady growth with seasonal peaks

## Data Source
Airbnb Listings 2016 (Kaggle)
https://www.kaggle.com/datasets/alexanderfreberg/airbnb-listings-2016-dataset
